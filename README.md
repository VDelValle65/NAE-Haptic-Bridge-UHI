# NAE-Haptic-Bridge-UHI
NAE™ by Neuro-Arts for Seniors, LLC is a scientific protocol mitigating synaptic pruning via ocular-motor stimulation. Using OpenAICodex, we engineered the Universal Haptic Interface (UHI): a core proprietary hardware component facilitating biaxial rotary modulation as part of an ocular-motor stimulation protocol for geriatric care.
